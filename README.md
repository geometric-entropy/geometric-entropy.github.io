# Geometric Entropy Project Page

Static project page for:

https://geometric-entropy.github.io/

## Deploy

1. Clone or open the GitHub Pages repository:

   ```bash
   git clone https://github.com/geometric-entropy/geometric-entropy.github.io.git
   ```

2. Copy everything in this `project_page/` directory into the repository root.
   The repository root should contain `index.html`, `style.css`, and `assets/`.

3. Commit and push:

   ```bash
   git add .
   git commit -m "Add project page"
   git push origin main
   ```

4. In GitHub, go to `Settings -> Pages`.
   Select `Deploy from a branch`, then choose `main` and `/ (root)`.

5. After GitHub finishes deployment, open:

   https://geometric-entropy.github.io/
